开发过程中积累的Terminal 命令，我常忘记的一些命令

启动WebService

  1. sudo apachectl start;
  2. sudo apachectl stop;
  3. sudo apachectl restart;

访问http://127.0.0.1/~username 或 http://ip/~username

ATOM

  1. runner 运行快捷键 ctrl+R

React-Native初始化

  1. react-native init AwesomeProject
  2. cd AwesomeProject
  3. react-native run-ios

反编译查看符号表

cd otool
. venv/bin/activate

otool -ov 得到属性和变量
nm -u 得到mach-o中的符号表
otool -L 查看使用了那些库

检测项目中是否含有IDFA

grep -r advertisingIdentifier .

HEXO命令

  1. hexo clean 清理
  2. hexo g 编译
  3. hexo s 起本地服务
  4. hexo d 部署到GitHub

显示隐藏文件

defaults write com.apple.finder AppleShowAllFiles -bool true
