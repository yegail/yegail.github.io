今天星期四，天气晴

Aug 3, 2017 | 随心生活

这段时间总感觉时间一晃就过了，上班的时间忙下这里忙下那里就到下班的时间了。下班回去的路上又想不起今天又做了什么。

这两天早上到办公室就把这天准备要做的事情列出来，到快下班的时候看看哪些做了哪些没做。虽然每天都有一些没有做的事情，但内心的焦虑感在减少。写这篇是今天的一项任务，在没有写计划的时候我肯定不会在快要下班的时候还来写一篇博客。算是有一点成就了。

今早脑袋里在构思小说的开头。
“在经过一个噩梦连连的夜晚后，她拖着疲惫的身体洗漱、逗狗、出门赶公交。。。”
如此日常的开头，以至我都无法想出小说的剧情。

看不惯别人多年前写的代码，因没有修改任务，已经被领导警告了几次不要随便动代码，测试不会测。我真的是手痒痒啊！

ThoughtWorks在这周六有iOS的活动，真的很想去，可是周末又要去买植物放到新房子里，又约了牙医。这几项在时间上有点冲突，我想我还是能抽出时间去参加那个iOS活动。

目前我的电脑更新了macOS High Sierra beta，Xcode也是9.0beta4，手机更成了iOS11 beta4。今天准备提交app到iTunesConnect，构建版本邮件中出现工具链错误：

Invalid Toolchain - New apps and app updates must be built with the public (GM) versions of Xcode 6 or later, macOS, and iOS SDK. Don’t submit apps built with beta software including beta macOS builds.

我还是把电脑刷回Sierra吧。

今晚去练车，看看什么时候考试，bless me下次考过。
